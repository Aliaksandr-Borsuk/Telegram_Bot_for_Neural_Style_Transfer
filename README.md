# Telegram_Bot_for_Neural_Style_Transfer 
    
Автор бота: @badger_by (stepik id:  327028887)
  
## Что умеет этот бот ?
Этот Telegram бот перносит стиль одной картинки на другую, с помощью алгоритма NST (Neural Style Transfer).
